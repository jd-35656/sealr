# API Reference

Complete Python API documentation for sealr.

## Overview

The sealr Python API provides programmatic access to all functionality. It's designed to be simple, secure, and intuitive for developers who want to integrate sealr into their applications.

## Installation

```bash
pip install sealr
```

See [Installation Guide](../getting-started/installation.md) for other installation options.

## Core Module

The main entry point for programmatic access to sealr functionality.

::: sealr
    options:
      show_root_heading: true
      show_root_full_path: false
      show_source: false
      heading_level: 3

## Exceptions

Custom exceptions raised by sealr operations.

::: sealr.exceptions
    options:
      show_root_heading: true
      show_root_full_path: false
      show_source: false
      heading_level: 3
